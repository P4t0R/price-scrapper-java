## Instalación

```bash
#Clona el repositorio
git clone https://github.com/P4t0R/price-scrapper-java

#Compila el projecto
mvn clean install

#Ejecuta el projecto 
mvn spring-boot:run

#Ejecuta el projecto desde `.jar`
java -jar target\*jar*
```

## Linux

```bash
#Instalar java
sudo apt install openjdk-17-jdk

#Seleccionar version java
sudo update-alternatives --config java
```
